# PolyPreview

A previewer for various multimedia.

A revamp of
[VideoPreviewer](https://github.com/Not-A-Normal-Robot/VideoPreviewer).

## Features

- Supports images, videos, audio, and iframes.
- Works offline using Service Workers
- Aggressive minification using Closure Compiler

## Building Dependencies
- Java (a minimum of Java 11)
- NPM or PNPM

## Building
1. Run `npm run build` in your terminal. Or `pnpm` instead of `npm` if you use that.
2. The output is in `dist/`.
